# Zeal

I was unable to find a heat map habit tracker that met my criteria, so I made my own!

![ExampleImage](./example.png)

## Installation

Clone the repo

```bash
git clone git@github.com:LarryBaer/Zeal.git
```

Install packages

```bash
npm install
```

Start local server

```bash
npm start
```
